# вҡҷпёҸ Installation & Setup

## 1пёҸвғЈ Clone the repository

```bash
git clone https://github.com/yourusername/know-your-calories.git
cd know-your-calories
```

---

## 2пёҸвғЈ Install backend dependencies

```bash
cd backend
npm install
```

---

## 3пёҸвғЈ Install frontend dependencies

```bash
cd ../frontend
npm install
```

---

## 4пёҸвғЈ Environment Variables

Create a `.env` file in the **backend folder** and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

## 5пёҸвғЈ Run the application

### Start backend

```bash
cd backend
npm run dev
```

### Start frontend

```bash
cd frontend
npm start
```

The app will run at:

```
Frontend: http://localhost:3000
Backend: http://localhost:5000
```
